Stockability

For this project we used API calls to retrieve stock information based on the stock ticker given by the user. Selected tickers are added to a table and there are two tabs with additional info including recent news. In addition to this a chart is displayed for the selected ticker. The user can also click a ticker in the table which will reload the chart for the selected ticker. The user has the ability to remove tickers from the chart as well.

Hangman

In this game the user has to guess the spelling of a champion from the game of League of Legends. The user is given the picture of a random champion and the game records correct an incorrect letter guesses. If the user runs out of guesses then the loss counter adds one. Conversely if the user gets the correct letters the wins counter adds one. In both scenarios of a win or a loss the next champion to guess is loaded to the page and the game starts again.

Magic Crystal Game

In this game a random number is generated and the user attempts to match that number by clicking on crystals. Each crystal is assigned a random value at the beginning of each round. When the user clicks the crystal that crystals value is added to the players number. If clicking a crystal causes the players number to be greater than the number they are trying to match then the player looses and a loss is added to the tally. If the player matches the number a win is added to the tally. In both cases a new round begins.

Trivia Game

In this game the player is asked questions about the United States such as the population or how many representatives the state has in the House of Representatives. The user has four responses to chose from and the correct response will appear in a random position for each question. When the quiz is started the user only has 30 seconds to answer the question before the next question starts. The user is notified of correct and incorrect responses. At the end of the quiz the user is shows a tally of correct and incorrect answers. The user has the option to start the quiz again.

GifTastic

On this site the user can type text into an input box to add a button to the page with the text entered. The user can then click on the new button to load 10 gifs form giphy.com. An API call is used to load the gifs. When the gifs are added they are the “still” version. The user can click the gif to animate it and click it again to stop the animation. In addition to this there is a gif options button and the user can click the gif to load the options. Using these options the user can pause, play backward or play the gif frame by frame.

Liri Bot

Liri Bot is a Node project. LIRI is like iPhone's SIRI. However, while SIRI is a Speech Interpretation and Recognition Interface, LIRI is a Language Interpretation and Recognition Interface. LIRI is a command line node app that takes in parameters and gives you back data. Axios powers the app and is used to send requests to Bands in Town, Spotify and OMDB APIs. A few meaningful data points are displayed to the user in the command line.
